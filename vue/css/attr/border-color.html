border-color ​

描述 ​

该 border-color 属性指定元素的边框颜色。用作在一个规则中确定整个边框颜色的快捷方式。

效果展示 ​

语法 ​

css
border-color: <color>{1,4}

值 ​

  • <color> 指定要在所有边框上使用的颜色。这可以是分别表示上边框、右边框、下边框和左边框的一到四个值。
  • transparent 这将应用不可见的边框，但可以应用宽度。
  • inherit 继承
