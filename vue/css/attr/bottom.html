bottom ​

描述 ​

该 bottom 属性设置元素内容下边缘的位置，而不考虑框架和缩进的粗细。坐标计数取决于位置属性，它通常取相对（相对位置）或绝对（绝对位置）的值。

效果展示 ​

语法 ​

css
bottom: auto | <length> | <percentage>;

值 ​

  • <length> 浮点数，后跟绝对单位指示符（厘米、毫米、英寸、pt 或 pc）或相对单位指示符（em、ex 或 px）。
  • <percentage> 整数，后跟百分号 （%）。该值是父对象高度的百分比。
  • auto 默认位置，根据页面的常规 HTML 布局。
