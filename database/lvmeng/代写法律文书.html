# 代写法律文书

流程，委托书说明案情，要求，提供案件资料。书写文书，可以修改三次，周期三天，费用 200 以上

# 线下流程

  1. 客户说明案情，要求，提供案件资料。
  2. 谈价格
  3. 成交
  4. 律师书写文书（三日内）
  5. 交付给客户
  6. 客户查看文书后，如果感觉有不妥，可与律师进行沟通，律师可修改三次
  7. 修改周期每次 1 天
  8. 完成

# 线上流程

线上代写文书有两种方式

  1. 客户选则律师进行委托
  2. 客户说明案情，要求，提供案件资料。设置价格，发布委托。律师抢单
    1. 该方式可以由律盟官方直接接单（不通知律师抢单）

与客户的沟通方式

  1. 律盟委托订单内进行沟通
  2. //律盟私聊沟通
  3. //电话沟通

问题：

  1. 是否由客户指定律师
    1. 客户指定律师存在问题 律师接单及时性问题，
  2. 先支付，还是先沟通
    1. 先支付：优势 方便平台监官，必免线下交易。缺点 价格需要先指定好，不容易根据难易程度定价。
    2. 先沟通：优势 方便律师根据案件难易程度定价，有效保护用户利益。缺点 容易出现线下交易的情况。
  3. 交付形式
  4. 修改次数
  5. 周期
    1. 接单时间限制 3 小时
    2. 接单后交付时间限制 -超时-先提醒-后扣分
    3. 修改时间限制
    4. 交付后 2 天结束我也不知道
  6. 沟通方式我问一我

  1. 客户选，和广场
  2. 先支付
  3. 新窗口
  4. 文件
